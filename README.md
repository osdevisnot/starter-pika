# starter-pika

This project is a starter for modern applications using pika-web. To use this starter, ensure you have `degit` installed.

```
npm install -g degit
```

To create a new project based on this template:

```
degit osdevisnot/starter-pika my-app && $_ && node setup.js
```

> for optimal setup, use this starter as a module in monorepo setup based on https://github.com/osdevisnot/starter-monorepo
